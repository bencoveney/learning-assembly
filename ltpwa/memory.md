# Memory

## Sizes

| Size (bits) | Size (bytes) | Known as               |
| ----------- | ------------ | ---------------------- |
| 1           |              | bit                    |
| 8           | 1            | byte                   |
| 16          | 2            | word, short            |
| 32          | 3            | double-word, int, long |
| 64          | 4            | quad-word, long        |

Some names are reused depending on context, which makes things confusing.
